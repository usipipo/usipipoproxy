# WireGuard Agent IP Allocation (Go) - Implementation Guide

## Overview

The Go agent's `WireGuardClient` needs DB-first IP allocation with full rollback capabilities.

### Key Changes

1. **Remove** `getNextAvailableIP()` config-file scan
2. **Add** `IPAllocationService` with `ReserveIP()`, `ConfirmIP()`, `ReleaseIP()`
3. **Add** reconciliation loop (runs every 30s)
4. **Add** cross-platform locking (flock + Windows named mutex)
5. **Add** structured logging + Prometheus metrics

```go
type IPAllocationClient struct {
    baseURL    string
    apiKey     string
    client     *http.Client
    serverID   string
    logger     *log.Logger
    metrics    *MetricsCollector
}

func NewIPAllocationClient(baseURL, apiKey, serverID string) *IPAllocationClient {
    return &IPAllocationClient{
        baseURL:  baseURL,
        apiKey:   apiKey,
        serverID: serverID,
        client: &http.Client{
            Timeout: 15 * time.Second,
        },
        logger:  log.New(log.Writer(), "[IPAlloc] ", log.Flags()),
        metrics: metrics.NewCollector(serverID),
    }
}

// ReserveIP atomically reserves next free IP in DB
// Retries up to 3 times with exponential backoff on contention
func (c *IPAllocationClient) ReserveIP(
    ctx context.Context,
    keyName string,
) (*IPAllocation, error) {

    req := IPReserveRequest{
        ServerID:   c.serverID,
        VpnKeyName: keyName,
        VpnType:    "wireguard",
        RequestID:  uuid.New().String(),
    }

    var resp IPReserveResponse
    err := c.doRequest(ctx, "POST", "/wireguard/reserve-ip", req, &resp)
    if err != nil {
        return nil, err
    }

    return &IPAllocation{
        IPAddress:       net.ParseIP(resp.IPAddress),
        IPInt:           resp.IPInt,
        LeaseID:         resp.LeaseID,
        CIDR:            mustParseCIDR(resp.CIDR),
        LeaseExpiresAt:  resp.LeaseExpiresAt,
        PeerName:        keyName,
    }, nil
}

// ConfirmIP marks reservation as confirmed after kernel+config success
func (c *IPAllocationClient) ConfirmIP(
    ctx context.Context,
    ipAddress string,
    publicKey string,
) error {
    req := IPConfirmRequest{
        IPAddress: ipAddress,
        PublicKey: publicKey,
        RequestID: uuid.New().String(),
```

```go
func (c *WireGuardClient) CreatePeer(ctx context.Context, name string) (*WireGuardPeer, error) {
    // ──────────────────────────────────────────────────────────────
    // PHASE 1: Reserve IP from Central DB
    // ──────────────────────────────────────────────────────────────
    allocation, err := c.ipAllocClient.ReserveIP(ctx, name)
    if err != nil {
        c.metrics.IncIPAllocationFailure("reserve")
        return nil, fmt.Errorf("failed to reserve IP: %w", err)
    }
    c.metrics.ObserveIPAllocationLatency("reserve", time.Since(start))

    // Track correlation ID across all logs/requests
    ctx = context.WithValue(ctx, "request_id", allocation.RequestID)
    c.logger.Info("ip_reserved", "ip", allocation.IPAddress, "key", name)

    // ──────────────────────────────────────────────────────────────
    // PHASE 2: Generate keys (local, cannot fail easily)
    // ──────────────────────────────────────────────────────────────
    privateKey, err := wgtypes.GeneratePrivateKey()
    if err != nil {
        // Compensate: release DB reservation
        _ = c.ipAllocClient.ReleaseIP(ctx, allocation.IPAddress.String(), "keygen_failed")
        return nil, fmt.Errorf("keygen failed: %w", err)
    }
    publicKey := privateKey.PublicKey()
    psk, err := wgtypes.GenerateKey()
    if err != nil {
        _ = c.ipAllocClient.ReleaseIP(ctx, allocation.IPAddress.String(), "psk_failed")
        return nil, fmt.Errorf("pskgen failed: %w", err)
    }

    // ──────────────────────────────────────────────────────────────
    // PHASE 3: Configure Kernel via wgctrl
    // ──────────────────────────────────────────────────────────────
    peerConfig := wgtypes.PeerConfig{
        PublicKey:         publicKey,
        PresharedKey:      &psk,
        AllowedIPs:        []net.IPNet{{IP: allocation.IPAddress, Mask: net.CIDRMask(32, 32)}},
        ReplaceAllowedIPs: false,
    }
    config := wgtypes.Config{Peers: []wgtypes.PeerConfig{peerConfig}}

    if err := c.client.ConfigureDevice(c.interfaceName, config); err != nil {
        // Compensate: release DB + nothing to clean in kernel (op failed)
        _ = c.ipAllocClient.ReleaseIP(ctx, allocation.IPAddress.String(), "kernel_failed")
        c.logger.Error("kernel_configure_failed", "ip", allocation.IPAddress, "err", err)
        return nil, fmt.Errorf("kernel config failed: %w", err)
    }
    c.logger.Info("kernel_peer_added", "ip", allocation.IPAddress, "pubkey", publicKey.String())

    // ──────────────────────────────────────────────────────────────
    // PHASE 4: Atomically update config file
    // ──────────────────────────────────────────────────────────────
    serverDevice, err := c.client.Device(c.interfaceName)
    if err != nil {
        // Partial: kernel updated, need rollback
        _ = c.client.ConfigureDevice(c.interfaceName, wgtypes.Config{
            Peers: []wgtypes.PeerConfig{{PublicKey: publicKey, Remove: true}},
        })
        _ = c.ipAllocClient.ReleaseIP(ctx, allocation.IPAddress.String(), "kernel_rollback")
        return nil, fmt.Errorf("failed to get device info: %w", err)
    }

    clientConfig := c.generateClientConfig(
        privateKey.String(),
        allocation.IPAddress.String(),
        serverDevice.PublicKey.String(),
        psk.String(),
    )
    peerBlock := fmt.Sprintf("### CLIENT %s\n[Peer]\nPublicKey = %s\nPresharedKey = %s\nAllowedIPs = %s/32\n",
        name, publicKey.String(), psk.String(), allocation.IPAddress.String())

    if err := c.updateConfigFileAtomic(peerBlock); err != nil {
        // Compensate: remove from kernel, release DB
        _ = c.client.ConfigureDevice(c.interfaceName, wgtypes.Config{
            Peers: []wgtypes.PeerConfig{{PublicKey: publicKey, Remove: true}},
        })
        _ = c.ipAllocClient.ReleaseIP(ctx, allocation.IPAddress.String(), "config_failed")
        c.logger.Error("config_write_failed", "ip", allocation.IPAddress, "err", err)
        return nil, fmt.Errorf("config write failed: %w", err)
    }

    // ──────────────────────────────────────────────────────────────
    // PHASE 5: Confirm DB allocation (now fully committed)
    // ──────────────────────────────────────────────────────────────
    if err := c.ipAllocClient.ConfirmIP(ctx, allocation.IPAddress.String(), publicKey.String()); err != nil {
        // Non-fatal: kernel + config done. Log for manual reconciliation
        c.logger.Warn("db_confirm_failed_reconciliation_needed",
            "ip", allocation.IPAddress,
            "lease_id", allocation.LeaseID,
            "err", err)
        // Do NOT rollback kernel/config — we can recover later
    } else {
        c.logger.Info("allocation_confirmed", "ip", allocation.IPAddress, "key", name)
    }

    c.metrics.IncIPAllocationSuccess()
    c.metrics.SetAvailableIPs(c.getPoolSize() - 1) // decrement

    return &WireGuardPeer{
        PublicKey:  publicKey.String(),
        Name:       name,
        IPAddress:  allocation.IPAddress.String(),
        Config:     clientConfig,
    }, nil
}
```

## Reconciliation Loop

```go
// ReconciliationResult holds drift summary
type ReconciliationResult struct {
    DriftDetected      bool
    MissingInDB        []wgtypes.Peer
    StaleInDB          []VpnIpAllocation
    IPMismatches       []IPMismatch
    FixesApplied       int
    Errors             []error
}

// Reconcile scans kernel vs DB vs Config, heals drift
func (c *WireGuardClient) Reconcile(ctx context.Context) ReconciliationResult {
    start := time.Now()
    result := ReconciliationResult{}

    // Acquire lock (shorter TTL for reconciler)
    lock, err := c.acquireIPLockWithTimeout(5 * time.Second)
    if err != nil {
        c.logger.Warn("reconcile_skip_lock_failed", "err", err)
        return result
    }
    defer lock.Release()

    // 1. Snapshot kernel peers
    device, err := c.client.Device(c.interfaceName)
    if err != nil {
        c.logger.Error("reconcile_kernel_error", "err", err)
        return result
    }
    kernelPeers := make(map[string]wgtypes.Peer)
    for _, p := range device.Peers {
        kernelPeers[p.PublicKey.String()] = p
    }

    // 2. Parse config file
    configBlocks := parseConfigFile(c.configPath)
    configMap := make(map[string]ConfigBlock) // pubkey → block
    for _, b := range configBlocks {
        configMap[b.PublicKey] = b
    }

    // 3. Query DB for allocated IPs (including orphaned/unconfirmed)
    dbAllocs, err := c.db.GetActiveAllocations(ctx, c.serverID)
    if err != nil {
        c.logger.Error("reconcile_db_error", "err", err)
        return result
    }

    // 4. Build maps
    dbMap := make(map[string]VpnIpAllocation) // pubkey → allocation
    for _, a := range dbAllocs {
        dbMap[a.PublicKey] = a
    }

    // 5. Detect drift

    // A. Kernel peers not in DB → orphan
    for pubkey, peer := range kernelPeers {
        if _, inDB := dbMap[pubkey]; !inDB {
            result.MissingInDB = append(result.MissingInDB, peer)
            // Auto-heal: create DB record with status=active, log for audit
            c.healOrphan(ctx, pubkey, peer)
            result.FixesApplied++
        }
    }

    // B. DB entries not in kernel → stale
    for pubkey, alloc := range dbMap {
        if _, inKernel := kernelPeers[pubkey]; !inKernel && alloc.Status == "allocated" {
            result.StaleInDB = append(result.StaleInDB, alloc)
            // Auto-heal: mark as revoked (safe, kernel already missing)
            c.healStaleDBEntry(ctx, alloc.ID)
            result.FixesApplied++
        }
    }

    // C. IP mismatch (kernel ≠ DB)
    for pubkey, peer := range kernelPeers {
        if alloc, inDB := dbMap[pubkey]; inDB {
            if !peer.AllowedIPs[0].IP.Equal(alloc.IPAddress) {
                result.IPMismatches = append(result.IPMismatches, IPMismatch{
                    PublicKey: pubkey,
                    DBIP:      alloc.IPAddress,
                    KernelIP:  peer.AllowedIPs[0].IP,
                })
                // Kernel wins, update DB
                c.healIPMismatch(ctx, alloc.ID, peer.AllowedIPs[0].IP)
                result.FixesApplied++
            }
        }
    }

    // 6. Log run
    durationMs := int(time.Since(start).Milliseconds())
    c.db.LogReconciliationRun(ctx, c.serverID, kernelPeers, dbAllocs, result, durationMs)
    c.logger.Info("reconciliation_complete",
        "duration_ms", durationMs,
        "drift", result.DriftDetected,
        "fixes", result.FixesApplied)

    return result
}

// healOrphan: kernel peer exists but no DB record
func (c *WireGuardClient) healOrphan(
    ctx context.Context,
    pubkey string,
    peer wgtypes.Peer,
) {
    ipStr := peer.AllowedIPs[0].IP.String()
    c.logger.Warn("healing_orphan", "pubkey", pubkey[:8], "ip", ipStr)

    // Attempt to infer key_name from config block (if available)
    name := extractPeerNameFromConfig(c.configPath, pubkey)
    if name == "" {
        name = "orphan-" + pubkey[:8]
    }

    // Create DB record
    err := c.db.CreateOrphanAllocation(ctx, &VpnIpAllocation{
        ServerID:            c.serverID,
        IPAddress:           net.ParseIP(ipStr),
        IPInt:              extractLastOctet(ipStr),
        Status:             "allocated",
        AssignedToPeerName: name,
        LeaseExpiresAt:     time.Now().Add(30 * 24 * time.Hour),
    })
    if err != nil {
        c.logger.Error("heal_orphan_failed", "err", err)
    }
    c.metrics.IncDriftFixed("orphan")
}

// healStaleDBEntry: DB entry exists but kernel peer is gone
func (c *WireGuardClient) healStaleDBEntry(ctx context.Context, leaseID string) {
    c.logger.Warn("healing_stale_db", "lease_id", leaseID)
    err := c.db.MarkAllocationRevoked(ctx, leaseID, "kernel_missing")
    if err != nil {
        c.logger.Error("heal_stale_failed", "err", err)
    }
    c.metrics.IncDriftFixed("stale_db")
}

// healIPMismatch: kernel and DB disagree on IP
func (c *WireGuardClient) healIPMismatch(
    ctx context.Context,
    leaseID string,
    correctIP net.IP,
) {
    c.logger.Warn("healing_ip_mismatch",
        "lease_id", leaseID,
        "correct_ip", correctIP.String())
    err := c.db.UpdateAllocationIP(ctx, leaseID, correctIP)
    if err != nil {
        c.logger.Error("heal_mismatch_failed", "err", err)
    }
    c.metrics.IncDriftFixed("ip_mismatch")
}
```

## Agent Startup: Initialize IP Pool

If `vpn_ip_allocations` table is empty for a server (first boot), pre-populate:

```go
func (c *WireGuardClient) EnsureIPPoolInitialized(ctx context.Context) error {
    count, err := c.db.GetIPAllocationCount(ctx, c.serverID)
    if err != nil {
        return err
    }
    if count > 0 {
        return nil // already initialized
    }

    c.logger.Info("initializing_ip_pool", "server_id", c.serverID)

    // Parse network CIDR from config (or settings)
    _, cidr, err := c.getNetworkCIDR()
    if err != nil {
        return fmt.Errorf("cannot determine network CIDR: %w", err)
    }

    // Generate all /32 host IPs from .2 to .254
    baseIP := extractBaseIP(cidr.IP) // e.g., 10.88.88.
    start := 2
    end := 254

    // Batch insert 253 rows
    batchSize := 50
    for i := start; i <= end; i += batchSize {
        batchEnd := min(i+batchSize-1, end)
        allocs := make([]VpnIpAllocation, 0, batchEnd-i+1)
        for j := i; j <= batchEnd; j++ {
            ipStr := fmt.Sprintf("%s%d", baseIP, j)
            ip := net.ParseIP(ipStr)
            allocs = append(allocs, VpnIpAllocation{
                ServerID:    c.serverID,
                IPAddress:   ip,
                IPInt:       j,
                CIDR:        cidr.String(),
                Status:      "free",
            })
        }
        if err := c.db.BulkCreateIPAllocations(ctx, allocs); err != nil {
            return err
        }
    }

    c.logger.Info("ip_pool_initialized", "count", end-start+1)
    return nil
}
```

## Agent Shutdown: Graceful Cleanup

```go
func (c *WireGuardClient) GracefulShutdown(ctx context.Context) error {
    c.logger.Info("shutdown_started")
    // Mark self as "draining" in DB (skip new allocations)
    _ = c.db.SetAgentStatus(ctx, c.serverID, "draining")

    // Wait for in-flight allocations to finish (max 30s)
    timeout := time.After(30 * time.Second)
    ticker := time.NewTicker(500 * time.Millisecond)
    defer ticker.Stop()

    for {
        select {
        case <-ctx.Done():
            return ctx.Err()
        case <-timeout:
            c.logger.Warn("shutdown_timeout_force_exit")
            return nil
        case <-ticker.C:
            inflight := c.GetInFlightAllocations()
            if inflight == 0 {
                c.logger.Info("shutdown_clean")
                return nil
            }
            c.logger.Debug("shutdown_waiting", "inflight", inflight)
        }
    }
}
```

```python
# src/core/application/services/ip_allocation_service.py
class IPAllocationService:
    """Handles atomic IP reservation and lifecycle."""

    def __init__(self, session: AsyncSession):
        self.session = session

    async def reserve_next_free_ip(
        self,
        server_id: UUID,
        vpn_key_id: UUID,
        peer_name: str,
        vpn_type: str,
    ) -> IPAllocation:
        """
        Atomically reserve next free IP using SELECT FOR UPDATE SKIP LOCKED.
        This function must be called within a transaction.
        """
        # Lock the pool row to prevent concurrent scans (optional)
        # Actually we lock at IP row level, not pool, for better concurrency

        # Find first free IP (lowest numeric value)
        stmt = (
            select(VpnIpAllocationModel)
            .where(
                VpnIpAllocationModel.server_id == server_id,
                VpnIpAllocationModel.status == "free",
                VpnIpAllocationModel.ip_int.between(2, 254),
            )
            .order_by(VpnIpAllocationModel.ip_int.asc())
            .with_for_update(skip_locked=True)
            .limit(1)
        )
        result = await self.session.execute(stmt)
        allocation = result.scalar_one_or_none()

        if allocation is None:
            raise NoAvailableIPsError(f"No free IPs for server {server_id}")

        # Mark reserved
        allocation.status = "reserved"
        allocation.vpn_key_id = vpn_key_id
        allocation.assigned_to_peer_name = peer_name
        allocation.allocated_at = datetime.now(UTC)
        allocation.lease_expires_at = datetime.now(UTC) + timedelta(days=30)

        # Log
        log = VpnIpAllocationLogModel(
            server_id=server_id,
            vpn_key_id=vpn_key_id,
            action="reserve",
            ip_address=allocation.ip_address,
            from_status="free",
            to_status="reserved",
            actor_type="backend",
            actor_id="system",
        )
        self.session.add(log)

        await self.session.flush()  # write but don't commit yet

        return allocation.to_entity()

    async def confirm_allocation(
        self,
        lease_id: UUID,
        ip_address: net.IPv4Address,
    ) -> bool:
        """Mark reservation as confirmed after agent reports success."""
        result = await self.session.execute(
            update(VpnIpAllocationModel)
            .where(
                VpnIpAllocationModel.id == lease_id,
                VpnIpAllocationModel.status == "reserved",
            )
            .values(status="allocated", confirmed_at=datetime.now(UTC))
        )
        return result.rowcount > 0

    async def release_ip(
        self,
        lease_id: UUID,
        reason: str,
    ) -> bool:
        """Compensation: mark IP as free again."""
        result = await self.session.execute(
            update(VpnIpAllocationModel)
            .where(
                VpnIpAllocationModel.id == lease_id,
                VpnIpAllocationModel.status.in_(["reserved", "allocated"]),
            )
            .values(
                status="free",
                vpn_key_id=None,
                assigned_to_peer_name=None,
                released_at=datetime.now(UTC),
                released_by=reason,
            )
        )
        return result.rowcount > 0

    async def reconcile_server(self, server_id: UUID) -> ReconciliationReport:
        """Run full reconciliation for a server (called by agent)."""
        # Fetch kernel state via agent, then:
        # - Detach mismatches
        # - Log to vpn_ip_reconciliation_runs
        pass

    async def get_available_count(self, server_id: UUID) -> int:
        """Get count of free IPs."""
        result = await self.session.execute(
            select(func.count(VpnIpAllocationModel.id))
            .where(
                VpnIpAllocationModel.server_id == server_id,
                VpnIpAllocationModel.status == "free",
            )
        )
        return result.scalar_one()
```

```python
@router.post("/reserve-ip")
async def reserve_ip(
    request: WireGuardReserveIPRequest,
    agent: Agent = Depends(get_authenticated_agent),
) -> WireGuardReserveIPResponse:
    """
    Reserve next free IP from pool for a new WireGuard peer.
    Idempotent: if called with same `peer_name`, returns existing reservation.
    """
    async with db.session() as session:
        # Check idempotency: does this peer already have a reservation?
        existing = await ip_allocation_repo.get_by_peer_name(
            server_id=agent.server_id,
            peer_name=request.vpn_key_name,
        )
        if existing and existing.status in ("reserved", "allocated"):
            return WireGuardReserveIPResponse(
                ip_address=str(existing.ip_address),
                ip_int=existing.ip_int,
                lease_id=existing.id,
                cidr=existing.cidr,
                lease_expires_at=existing.lease_expires_at,
            )

        # Validate vpn_type == wireguard
        if request.vpn_type != "wireguard":
            raise HTTPException(400, "Only WireGuard supports IP reservation")

        # Reserve
        allocation = await ip_service.reserve_next_free_ip(
            server_id=agent.server_id,
            vpn_key_id=uuid4(),  # Temporary; updated when key created
            peer_name=request.vpn_key_name,
            vpn_type=request.vpn_type,
        )

        await session.commit()
        return WireGuardReserveIPResponse.from_orm(allocation)
```

## Runbook: Operator Tasks

### Task: "IP Pool Exhausted"

**Symptoms:**  
`NoAvailableIPsError` in agent logs, health check shows `ip_pool.exhausted=true`

**Steps:**
1. Check current utilization: `SELECT * FROM vw_ip_pool_summary WHERE server_id='...'`
2. If `allocated_count ≈ 252`, pool exhausted → options:
   - **Cleanup stale**: `SELECT reclaim_stale_ips(grace_days => 7)` (reclaim leases older than 37 days)
   - **Expand range**: Update server config, run `ALTER TABLE vpn_ip_allocations ADD ...` (requires migration)
   - **Manual release**: Find unused keys (`SELECT * FROM vpn_keys WHERE last_seen_at < NOW() - INTERVAL '90 days'`)
3. Monitor after action.

### Task: "Drift Detected"

**Symptoms:**  
Reconciliation reports `drift_detected=true`

**Investigation:**
```sql
-- Check what drifted
SELECT * FROM vw_wireguard_drift_report WHERE server_id='...';

-- Review reconciliation run details
SELECT * FROM vpn_ip_reconciliation_runs
WHERE server_id='...'
ORDER BY started_at DESC LIMIT 5;
```

**Manual Fix (if needed):**
```bash
# Force re-sync of specific key
$ usipipo-repair wg-ip --key-id=abc123 --mode=reconcile
```

### Task: "Duplicate IP Detected"

**Symptoms:**  
`wg show` config has duplicate AllowedIPs, users report connection conflicts

**Steps:**
```sql
-- Find which IP is duplicated
SELECT ip_address, COUNT(*) c FROM vpn_ip_allocations
WHERE status='allocated' GROUP BY ip_address HAVING COUNT(*) > 1;

-- Find conflicting keys
SELECT k.name, k.ip_address
FROM vpn_keys k
WHERE k.ip_address IN ('duplicate_ip');
```

**Root cause:** Usually reconciliation bug or agent crash mid-allocation. Roll back one key (older allocation). Contact engineering.

## FAQ

**Q: What if backend DB is down?**  
A: Agent fails allocation with `503 Service Unavailable`. No fallback to config-file scan (to prevent duplicates). Queue for retry.

**Q: Can I pre-allocate an IP for a user before key creation?**  
A: Yes, via admin API: `POST /admin/vpn-keys/reserve-ip`.

**Q: Does this work across multiple agents on the same server?**  
A: Yes. `flock` (Unix) / named mutex (Windows) protects within host. DB row locks protect across all agents.

**Q: How long does lease last by default?**  
A: 30 days. Auto-reclaimed after 37 days (30 + 7 grace). Tunable per server via `lease_ttl_seconds`.

**Q: Where is the source of truth for IP assignments?**  
A: Database `vpn_ip_allocations` table. Config file and kernel are derivable from DB.

**Q: What if reconciliation makes a wrong fix?**  
A: All fixes are logged to `vpn_ip_allocation_log`. Use audit trail to investigate and manually reverse if needed.

## Appendix: Migration Steps (Zero-Downtime)

1. Deploy DB migration (adds tables + nullable columns) → zero impact
2. Deploy backend code with `ENABLE_DB_IP_ALLOCATION=false` (default)
3. Pre-populate `vpn_ip_allocations` for existing servers (one-time script)
4. Enable feature flag per server in DB: `UPDATE vpn_servers SET ip_allocation_mode = 'db' WHERE ...`
5. Deploy agent v2 (reads `ENABLE_DB_IP_ALLOCATION`)
6. Monitor `wg_ip_allocations_total{result="success"}` ↗, `failure` ↘
7. After 24h stable, enable globally
