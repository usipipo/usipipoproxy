# uSipipo — Monorepo

Monorepo de uSipipo. Todos los proyectos en un solo lugar.

## Estructura

- `apps/` — Aplicaciones desplegables (backend, bot, app, appweb)
- `packages/` — Librerías compartidas (common, agent)
- `infra/` — Docker, CI/CD, Terraform, despliegues
- `docs/` — Documentación del proyecto
